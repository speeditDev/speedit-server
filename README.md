<h3> :heavy_check_mark: About </h3>  

> **직장인들의 성장을 위한 도서 문장 기록 & 공유 서비스 "스피딧"**
> 
> 독서를 통해서 성장하고 싶은 직장인들이 도서 문장을 기록하고 다양한 직무/직업별로 공유를 통해서  
> 책 문장을 통해서 다양한 직업의 사람들과 소통 & 성장하는 서비스입니다.
